🌟 Calculator Project

📌 About the Project

This project was created as part of my Frontend Development Internship.

It is a simple and responsive Calculator application built using HTML, CSS, and JavaScript. The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division through an interactive and user-friendly interface.

- Clear and delete functionality
